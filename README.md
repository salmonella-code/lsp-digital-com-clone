
# Website Lembaga Sertifikasi Profesi
Website lembaga sertifikasi profesi ini dibangun menggunakan bootstrap dan laravel

## Requitment
- PHP Verisi >= 7.4
- Apache 2
- Composer 2

## Installation

Buka file projek kemudian sesuaikan konfigurasi database yang ada di file .env

jalankan migration
```bash
  php artisan migrate --seed
```

jalankan projek laravel
```bash
  php artisan serve
```

buka web browser
```bash
  Default url: 127.0.0.1:8000
```
